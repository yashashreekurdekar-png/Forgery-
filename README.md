# Image Forgery Detection System

## Introduction
This repository hosts the documentation for the Image Forgery Detection System project, which aims to identify and detect forged images using advanced machine learning techniques.

## Features
- Detects various types of image forgery, including splicing, copy-move, and image retouching.
- Utilizes state-of-the-art algorithms for feature extraction and classification.
- Provides performance metrics and analysis of detection accuracy.

## Installation
To set up the Image Forgery Detection System, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/yashashreekurdekar-png/Forgery-
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
- Place the image you want to analyze in the `images` directory.
- Run the detection script:
   ```bash
   python detect_forgery.py
   ```
- The results will be saved in the `results` directory.
